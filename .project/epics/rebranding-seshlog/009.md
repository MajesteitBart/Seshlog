# Task: Generate app icons from SVG

## Description

Generate all required application icons from the Seshlog SVG logo (`docs/seshlog_icon.svg`). This includes PNG files at various sizes for different platforms and ICO/ICNS files for Windows/macOS.

## Acceptance Criteria

- [ ] Generated `icons/32x32.png` (taskbar, small icon)
- [ ] Generated `icons/128x128.png` (standard app icon)
- [ ] Generated `icons/128x128@2x.png` (256x256, Retina displays)
- [ ] Generated `icons/icon.ico` (Windows, multi-resolution: 16, 32, 48, 64, 128, 256)
- [ ] Generated `icons/icon.icns` (macOS, for future builds)
- [ ] Generated `frontend/public/favicon.ico` (web views)
- [ ] Icons render crisply without aliasing artifacts
- [ ] Logo silhouette is recognizable at 16x16 size

## Technical Details

### Source File
- `docs/seshlog_icon.svg` - Seshat-inspired hieroglyph design

### Output Directory
- `frontend/src-tauri/icons/` - Tauri app icons
- `frontend/public/` - Web favicon

### Generation Commands (ImageMagick)

```bash
# Navigate to frontend/src-tauri directory
cd frontend/src-tauri

# Generate PNGs
convert -background white ../../docs/seshlog_icon.svg -resize 32x32 icons/32x32.png
convert -background white ../../docs/seshlog_icon.svg -resize 128x128 icons/128x128.png
convert -background white ../../docs/seshlog_icon.svg -resize 256x256 icons/128x128@2x.png

# Generate Windows ICO (multi-resolution)
convert -background white ../../docs/seshlog_icon.svg -define icon:auto-resize=256,128,64,48,32,16 icons/icon.ico

# Generate macOS ICNS (requires iconutil or png2icns)
# Alternative: Use online converter or Sharp

# Generate favicon
convert -background white ../../docs/seshlog_icon.svg -resize 32x32 ../public/favicon.ico
```

### Alternative: Using Sharp (Node.js)

```javascript
const sharp = require('sharp');

// Generate PNGs
await sharp('docs/seshlog_icon.svg').resize(32, 32).png().toFile('icons/32x32.png');
await sharp('docs/seshlog_icon.svg').resize(128, 128).png().toFile('icons/128x128.png');
await sharp('docs/seshlog_icon.svg').resize(256, 256).png().toFile('icons/128x128@2x.png');
```

### Notes
- The SVG has a white background (#FFFFFF in the rect element) so `-background white` ensures consistency
- If icons look poor at small sizes, may need to create a simplified version of the logo
- Test the 16x16 size in the ICO to ensure the silhouette is recognizable
