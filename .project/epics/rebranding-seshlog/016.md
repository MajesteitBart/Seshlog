### Implementation

```bash
# Copy and resize from existing icons
cd frontend

# For collapsed logo (small, ~40px wide)
# Use icon_32x32.png or icon_48x48.png as base
cp src-tauri/icons/icon-48.png public/logo-collapsed.png

# For full logo
cp src-tauri/icons/icon.png public/logo.png
```

Or use ImageMagick to resize:
```bash
convert src-tauri/icons/icon.png -resize 48x48 public/logo-collapsed.png
convert src-tauri/icons/icon.png -resize 256x256 public/logo.png
```
